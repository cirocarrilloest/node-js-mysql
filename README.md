# actividad de crud

## CRUD con node.js, mysql

## instalacion

#### shell

- npm i express mysql2
- npm i dotenv
- npm i -D nodemon
- npm install cors

## ejecutar

#### shell

- node src/server.js

## iniciar

- npm start

## configuracion del .nav

esta como .nav-example solo eliminar esta parte "-example" y poner el la contraseña del root del MySQL, el script del la base de datos esta en la carpeta solo de copiarlo o abrirlo
